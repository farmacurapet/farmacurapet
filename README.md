# FarmacuraPet 🐾
**Farmácia Solidária para Pets**

Este projeto visa conectar doadores de medicamentos veterinários a tutores e protetores que precisam de ajuda, promovendo a economia circular e o bem-estar animal.

## 🚀 Tecnologias
- Python
- Kivy (Interface Mobile)
- SQLite (Banco de dados local)
- Pydroid 3 (Ambiente de desenvolvimento)

## 📋 Status: Quest 3 (Em desenvolvimento)
Atualmente focado na persistência de dados e listagem de doações.
